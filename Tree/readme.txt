
Tree takes two input arguments the first being the input file name and the second being
the output file name.


The input file needs to be in the same directory as the Tree class file when
running, Tree will create the output file using the name given in the argument,
The output file is created in the same directory as the source file.

output file will be overwriten with each run of the program.
 
The Tree code was created such that it functions under all requirements and constraints
found in the Requirements&Constraints.txt file
