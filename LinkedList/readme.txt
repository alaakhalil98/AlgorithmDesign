
LinkedList takes two input arguments the first being the input file name and the second being
the output file name.


The input file needs to be in the same directory as the Source files file when
running, LinkedList will create the output file using the name given in the argument,
The output file is created in the same directory as the source file.

output file will be overwriten with each run of the program.

The LinkedList is designed to function under all requirements and constrainsts 
listed in the Requiremnets&Constraints.txt file
